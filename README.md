# 💼 Job Search Website

A modern and responsive Job Search Website that helps users find jobs quickly and easily. The platform allows users to browse job listings, view detailed job descriptions, and explore opportunities through a clean and user-friendly interface.

This project focuses on fast search functionality, responsive design, and a simple user experience for job seekers.

# 🚀 Features

🔍 Real-time Job Search – Quickly search for jobs using keywords.

📄 Job Listings Page – Displays available job opportunities.

🧾 Job Details Page – View full job descriptions and information.

⚡ Fast & Lightweight – Built with simple and efficient frontend technologies.

📱 Responsive Design – Works smoothly on desktop, tablet, and mobile devices.

🎨 Clean User Interface – Simple layout for easy navigation.

# 🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Search functionality and interactivity
